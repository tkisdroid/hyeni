MemoSection DOM landmarks (from 05.5-01 implementation):
- `role="log" aria-label="메모 대화"` — bubble container
- `role="article"` per bubble with `aria-label="{sender} {relative_time}에 보낸 메시지: {content}"`
- `role="separator"` for date pill
- `role="status" aria-label="메모 화면이 새로워졌어요"` for onboarding toast
- `role="alert" aria-label="메시지 전송 실패"` for send-failure toast
- composer input has `aria-label="메모 입력"` (parent) or `"답글 입력"` (child)
- send button has `aria-label="메시지 보내기"`
- retry button has `aria-label="메시지 전송 다시 시도"`
- localStorage key: `memoOnboardingV2Seen` (value '1')
