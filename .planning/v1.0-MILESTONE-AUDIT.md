## Recommendation

**Status: `tech_debt`**

All 28 v1 REQs closed, all 5 phases complete, zero out-of-scope violations, zero boundary breaches on `src/App.jsx`, all migrations reversible, baselines intact. The milestone **met its stated DoD** (9 audit findings + SOS-01 scope expansion = 28 REQs, all closed).

However, ship-readiness carries explicit caveats that prevent a clean `passed`:

1. **APK-level remote-listen is still source-authored, not runtime-shipped** — Play Store policy risk until v1.1 native-deploy closes FGS wiring.
2. **Memos DROP deferred by design** (shadow-running DoD, not cutover) — MEMO-CLEANUP-01 opens as the first v1.1 ticket.
3. **8 human-verification items remain** (ES256 200 happy-path, PairingModal TTL visual, press-hold timing, FGS persistent notification) — all programmatic evidence in place; only in-browser / on-device observation remains.
4. **push_idempotency TTL cron unowned** — growth unbounded long-term.

**Recommendation:**
- **Close v1.0 milestone as `tech_debt` (shipped with documented caveats).**
- **Open v1.1 milestone immediately with native-deploy as Phase 1.**
- Execute human-verification items as natural-usage observation during v1.1 planning window.
- Gate MEMO-CLEANUP-01 behind ≥30-day shadow-parity expiry (~2026-05-21).
- Surface push_idempotency TTL cron to v1.1 backlog (deferred tag: `v2-observability` if v1.1 is too full).

**Safety-critical paths are live and verified:**
- Push gateway: ES256 verified in-function, idempotent, observable (`pending_notifications.delivery_status`).
- Realtime: 7 per-table channels, 6 tables REPLICA IDENTITY FULL, publications active.
- Pair security: 48h TTL + parent-only regenerate + child DELETE-deny + parent-only zombie unpair (RLS-enforced, 3/3 smokes PASS).
- SOS audit: immutable `sos_events` + dedup_key + 5s server cooldown + 60s receiver LRU.
- Remote listen: audit-row-before-capture + kill switch + persistent indicator + beforeunload/pagehide cleanup.

The core-value contract — *"부모가 보낸 SOS/알림이 아이에게, 아이가 보낸 꾹이 부모에게, 앱이 닫혀 있어도 반드시 도달한다"* — is delivered at the Web/Supabase layer. The Android native layer ships as v1.1.
